📚 原理說明

什麼是合併排序？

合併排序是一種分而治之的排序算法，採用遞歸方式將陣列分成兩半，分別排序後再合併，具有穩定性和 O(n log n) 的時間複雜度。

核心概念

  • 分散：將陣列遞歸分成最小單位（單個元素）
  • 排序：在合併過程中進行比較和排序
  • 合併：將已排序的子陣列合併成更大的有序陣列
  • 名稱由來：通過不斷合併有序子序列來完成排序

演算法步驟

1. 將陣列分成兩半
2. 遞歸對左半部分排序
3. 遞歸對右半部分排序
4. 合併兩個已排序的子陣列
5. 重複直到整個陣列排序完成

時間複雜度分析

情況 時間複雜度 說明
最好情況 O(n log n) 已排序的陣列
平均情況 O(n log n) 隨機陣列
最壞情況 O(n log n) 完全逆序的陣列
空間複雜度 O(n) 需要額外的臨時陣列用於合併

優缺點

  • ✓ 優點：穩定排序、時間複雜度穩定、適合大數據
  • ✗ 缺點：需要額外空間、不適合鏈表排序
