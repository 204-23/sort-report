# 氣泡排序法 (Bubble Sort)  
原理簡述：氣泡排序是最基礎的交換排序。它重複走訪要排序的數列，一次比較兩個相鄰的元素，如果兩者的順序錯誤（如左大於右）就交換過來。每一輪完整的走訪後，數列中最大的元素會如同氣泡一樣「浮」到最右端。  
複雜度分析：時間複雜度： 平均與最差皆為 $O(n^2)$。若加入旗標（Flag）優化，最佳情況（已排序）可達 $O(n)$。空間複雜度： $O(1)$，僅需常數級額外空間。穩定性： 穩定排序（Stable）。
